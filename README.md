# 🎰 cassino-slot

Projeto de estudo de um cassino online — **fichas fictícias, sem dinheiro real**.

> ⚠️ Cassino com dinheiro real no Brasil é atividade regulada (Lei 14.790/2023) e
> exige autorização federal. Este repositório é apenas educacional.
